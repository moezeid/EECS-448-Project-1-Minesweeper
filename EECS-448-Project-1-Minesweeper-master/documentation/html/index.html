Pysweeper Documentation
This is documentation for Minesweeper in Python. This project was first created by the Code Chefs but has now been extended by Team FourFour8. Additions include:
  • A scoring system that scores and records winning scores.
  • A toggleable cheat board that reveals all mines and numbered spaces at the user's will.
  • Sound effects to intensify the game
